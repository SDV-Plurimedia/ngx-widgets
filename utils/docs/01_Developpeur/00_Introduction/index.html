Guide d'utilisation: Développeur

Ce guide a pour but d'aider les nouveaux développeur à comprendre LeaPulse

Nous essayerons au maximum de tenir cette documentation à jour, mais pour cela nous avons besoin de vous!

Pour chaque développement que vous faites sur cette plateforme, pensez à tenir la documentation à jour. ( la doc se trouve au format Markdown dans le repertoire /documentation/docs/ découpé dans les dossiers 01_Developpeur et 02_Utilisateur )

S'il y a un manque de précision dans une documentation ou des informations erronées Merci de nous tenir informés.

Pour installer votre environnement vous pouvez:

  • soit installer docker (méthode préférée sous linux) et lancer 'docker-compose up',
  • soit utiliser virtualbox avec notre image de développement (avec les identifiants leapulsedev/leapulsedev)
